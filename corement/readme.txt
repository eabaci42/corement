Corement is a modern, self-hosted comment system designed to replace the default WordPress comments with a more engaging and feature-rich experience. Built with security, performance, and user experience in mind.

== Installation ==

1. Upload the `corement` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Corement' in your WordPress admin menu to configure settings
4. Use the `[corement]` shortcode to display comments, or enable auto-append in settings

= Automatic Installation =

1. Go to Plugins > Add New in your WordPress admin
2. Search for "Corement"
3. Click "Install Now" and then "Activate"

== Frequently Asked Questions ==

= Is Corement a replacement for Disqus? =

Yes! Corement is designed as a self-hosted alternative to Disqus, giving you full control over your comment data while providing modern features like reactions and voting.

= Does it work with my theme? =

Corement is designed to work with any WordPress theme. It has special optimizations for Madara and WP-Manga themes, and automatically adapts to dark/light mode preferences.

= Can guests comment without registration? =

Yes, guest commenting is supported. Guests can enter their name and email to comment, with optional moderation approval.

= How secure is Corement? =

Corement includes comprehensive security features including XSS/CSRF protection, rate limiting, spam detection, and advanced blacklist management. All user inputs are sanitized and validated.

= Can I import existing WordPress comments? =

Currently, Corement works alongside WordPress comments. You can disable the default comment system and use only Corement, but import functionality is planned for future releases.

= Does it support media uploads? =

Yes, users can upload images and GIFs with their comments. File types and sizes are configurable, with automatic optimization and security scanning.

== Privacy Policy ==

Corement stores comment data locally on your WordPress site. No data is sent to external services. User IP addresses are stored for security and moderation purposes and can be configured to be automatically deleted after a specified period.
